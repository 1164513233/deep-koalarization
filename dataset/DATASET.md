# *#dataset*

## Getting the images from Imagenet

```bash
python3 -m dataset.download <args>
```

(use `-h`) to see the available options

## Resizing

```bash
python3 -m dataset.resize <args>
```

(use `-h`) to see the available options

## Filtering

Run:

```bash
python3 -m dataset.filters <args>
```

(use `-h`) to see the available options
